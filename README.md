# Stock Trend Analysis using Python

## Project Overview
This project analyzes stock price trends using moving averages and a simple rule-based strategy.

## Key Features
- Data cleaning and preprocessing
- Moving average trend analysis (50-day & 200-day)
- Buy/sell signal generation
- Strategy vs buy-and-hold comparison
- Risk metrics (volatility, Sharpe ratio, drawdown)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Insights
- Moving averages help identify overall trends
- Strategy performs better during trending markets
- Risk metrics are important to evaluate performance

## Dataset
Historical stock data from Kaggle
https://www.kaggle.com/datasets/borismarjanovic/price-volume-data-for-all-us-stocks-etfs?resource=download
